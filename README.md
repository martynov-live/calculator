# 🧮 Калькулятор

Калькулятор в стиле iOS: тёмная тема, круглые кнопки, плавные нажатия. Полностью повторяет логику и внешний вид системного калькулятора Apple. Работает в браузере без установки.

🌐 **[Открыть приложение](https://martynov-live.github.io/calculator/)**

## Возможности

- Базовые операции: сложение, вычитание, умножение, деление
- Проценты, смена знака (±), сброс (AC/C)
- **Строка истории** текущего выражения над результатом
- **Управление с клавиатуры**: цифры, операторы, `Enter` (=), `Esc` (сброс), `Backspace`
- Корректная обработка дробей и длинных чисел
- Адаптивная вёрстка под мобильные устройства

## Технологии

- **HTML5** — разметка интерфейса
- **CSS3** — система переменных, grid-раскладка кнопок, анимации нажатий, шрифты Inter / SF Pro Display
- **Vanilla JavaScript** — вся логика вычислений и обработка клавиатуры (без фреймворков)

## Запуск локально

Откройте `index.html` в любом браузере — никаких зависимостей не требуется.
